# Análise de Dados com Python — Monitoria PACCE

## Introdução à Análise de Dados e Desenvolvimento de Competências Computacionais

Este repositório reúne materiais, estudos, exercícios, exemplos práticos, projetos, resoluções e atividades desenvolvidas durante minha atuação como monitor da disciplina de **Análise de Dados com Python** no programa **PACCE**.

As atividades aqui organizadas têm como foco a introdução aos conceitos fundamentais de programação e análise de dados, utilizando a linguagem Python como ferramenta principal para manipulação, tratamento, visualização e interpretação de dados.

Além da elaboração de materiais de apoio e atividades práticas, a monitoria teve como objetivo auxiliar os estudantes no desenvolvimento do pensamento computacional, na resolução de problemas e na compreensão das principais técnicas utilizadas em projetos de análise de dados.

---

# Objetivos do Repositório

* Organizar os materiais produzidos durante a monitoria;
* Disponibilizar conteúdos de apoio para estudo de Python aplicado à análise de dados;
* Registrar atividades, exemplos e exercícios desenvolvidos ao longo da monitoria;
* Auxiliar estudantes na aprendizagem de programação e manipulação de dados;
* Servir como espaço de consulta, revisão e prática.
